# 🎬 Netflix Content Analysis Dashboard

## 📌 Project Overview

This project analyzes Netflix's content library using **Python** and **Power BI** to uncover insights about movies, TV shows, genres, ratings, countries, and content growth over time.

The goal of this project is to demonstrate data cleaning, exploratory data analysis (EDA), data visualization, and dashboard development skills for a real-world dataset.

## 🎯 Objectives

- Analyze Netflix's content library.
- Compare Movies and TV Shows.

## 🔍 Key Business Insights

- Movies make up the majority of Netflix's content library.
- The United States is the largest content-producing country.
- International Movies, Dramas, and Comedies are among the most common genres.
- Netflix rapidly expanded its content library between 2016 and 2019.
- TV-MA and TV-14 are the most common content ratings.
- Most movies have a duration between 80 and 120 minutes.
